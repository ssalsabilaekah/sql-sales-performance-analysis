# Sales Performance & Business Insight Analysis (SQL)

## Overview
This project demonstrates an end-to-end SQL workflow to analyze sales performance, customer behavior, and regional trends using cleaned transactional data.

## Business Objectives
- Evaluate overall sales and profit performance
- Identify top customers and products
- Analyze monthly sales trends
- Compare performance across regions and markets
- Apply advanced SQL techniques for business insights

## Dataset
- Source: Excel file (cleaned and standardized)
- Format: CSV
- Data includes sales, profit, customer, product, region, and time dimensions

## Tools & Technologies
- MySQL
- MySQL Workbench
- Microsoft Excel
- GitHub

## Key SQL Analyses
- Total Sales & Profit Metrics
- Monthly Sales Trend
- Sales by Region
- Top Customers by Revenue
- Customer Ranking using Window Functions
- Discount Impact Analysis

## Query Results
Query outputs are provided as screenshots to validate SQL logic and results.
Please refer to the `screenshots` folder.

## Sample Business Insights
- A small percentage of customers contribute significantly to total revenue
- Sales performance shows clear trends across different time periods
- Certain regions dominate overall sales
- Higher discounts do not always lead to higher profit
